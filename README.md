# Delhi International School

Premium React + Vite landing page for Delhi International School, designed around a modern high-impact school brand direction.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Deployment

The app builds to `dist/` and is ready for static deployment on platforms like GitHub Pages, Netlify, Vercel, or Cloudflare Pages.

## GitHub Pages

This repo includes a GitHub Actions workflow that builds and deploys the site to GitHub Pages whenever code is pushed to `main`.
